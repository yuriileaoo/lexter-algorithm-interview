## Run Locally

Clone the project

```bash
  git clone https://github.com/yuriileaoo/lexter-algorithm-interview.git
```
Install dependencies

```bash
  yarn
```

Run locally server

```bash
  yarn dev
```

### - Observability

To run nvision, start server with

```bash
  yarn nvision
```

### - Running Tests

To run tests, run the following command

```bash
  yarn test
```
